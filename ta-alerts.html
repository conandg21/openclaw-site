Alert Architecture

Most traders don't have time to watch charts 24/7. Signal alerts solve this by triggering automated notifications when key patterns occur — then you can decide whether to act. Three output modes, each with tradeoffs:

Three Output Modes
  • Email (smtplib): Best for important signals and low frequency. Requires SMTP credentials, but ensures alerts reach you anywhere.
  • File log (append to .txt): Full audit trail with no dependencies. Works without email setup. Spin up a separate monitoring script to read the file.
  • Daily digest (one email per day): Aggregate all signals into a single email at market close. Less noise, better pattern spotting.

Recommended approach: Run file log always (cheap insurance), email only for high-priority signals (golden cross, BB squeeze breakout). Schedule daily digest for 4pm ET Mon-Fri to catch end-of-day opportunities.

Base Alert Engine

Start with this foundation. It handles logging, email dispatch, and configuration via environment variables (no hardcoded secrets):

Python 
import yfinance as yf
import pandas_ta as ta
import pandas as pd
import smtplib
import time
import os
from datetime import datetime, date
from email.mime.text import MIMEText
from email.mime.multipart import MIMEMultipart

# Configuration
ALERT_EMAIL = os.getenv('ALERT_EMAIL', 'you@example.com')
SMTP_HOST   = os.getenv('SMTP_HOST', 'localhost')
SMTP_PORT   = int(os.getenv('SMTP_PORT', '587'))
SMTP_USER   = os.getenv('SMTP_USER', '')
SMTP_PASS   = os.getenv('SMTP_PASS', '')
LOG_FILE    = 'signal_alerts.log'

def log_alert(message: str) -> None:
    '''Append alert to log file with timestamp.'''
    ts = datetime.now().strftime('%Y-%m-%d %H:%M:%S')
    line = f'[{ts}] {message}'
    print(line)
    with open(LOG_FILE, 'a') as f:
        f.write(line + '\n')

def send_email(subject: str, body: str) -> bool:
    '''Send alert email. Returns True on success.'''
    try:
        msg = MIMEMultipart('alternative')
        msg['Subject'] = subject
        msg['From'] = ALERT_EMAIL
        msg['To']   = ALERT_EMAIL
        msg.attach(MIMEText(body, 'plain'))

        with smtplib.SMTP(SMTP_HOST, SMTP_PORT) as server:
            server.starttls()
            if SMTP_USER:
                server.login(SMTP_USER, SMTP_PASS)
            server.send_message(msg)
        return True
    except Exception as e:
        log_alert(f'EMAIL FAILED: {e}')
        return False

Indicator Snapshot for One Ticker

Before alerting, you need clean data and computed indicators. This function fetches one year of daily OHLCV, computes SMA20/50/200, RSI, Bollinger Bands width, and MACD:

Python 
def get_snapshot(ticker: str) -> dict | None:
    '''Fetch and compute indicators for a single ticker.'''
    try:
        df = yf.download(ticker, period='1y', interval='1d', progress=False)
        if df.empty or len(df) < 210:
            return None
        df.columns = [c.lower() for c in df.columns]
        df = df[['open','high','low','close','volume']].dropna()

        df['sma20']  = ta.sma(df['close'], 20)
        df['sma50']  = ta.sma(df['close'], 50)
        df['sma200'] = ta.sma(df['close'], 200)
        df['rsi']    = ta.rsi(df['close'], 14)
        bb = ta.bbands(df['close'], 20, 2)
        df['bb_bw']  = bb[[c for c in bb.columns if 'BBB' in c][0]]
        macd = ta.macd(df['close'], 12, 26, 9)
        df['macd']   = macd[[c for c in macd.columns if c.startswith('MACD_')][0]]
        df['macd_s'] = macd[[c for c in macd.columns if c.startswith('MACDs_')][0]]
        df = df.dropna()

        cur  = df.iloc[-1]
        prev = df.iloc[-2]
        return {'cur': cur, 'prev': prev, 'df': df, 'ticker': ticker}
    except Exception:
        return None

Signal Detection Functions

Each detector runs on the current vs. previous bar, looking for a cross or threshold breach. Return a list of alert dicts with message, type, and priority:

Python — RSI Oversold/Overbought 
def check_rsi_alert(snap: dict, oversold: float = 30, overbought: float = 70) -> list:
    '''Detect RSI threshold crosses (entering oversold/overbought).'''
    alerts = []
    cur_rsi  = snap['cur']['rsi']
    prev_rsi = snap['prev']['rsi']

    if cur_rsi < oversold and prev_rsi >= oversold:
        alerts.append({
            'type': 'RSI_OVERSOLD',
            'msg':  f"{snap['ticker']} RSI entered oversold: {cur_rsi:.1f} (was {prev_rsi:.1f})",
            'priority': 'medium',
        })
    if cur_rsi > overbought and prev_rsi <= overbought:
        alerts.append({
            'type': 'RSI_OVERBOUGHT',
            'msg':  f"{snap['ticker']} RSI entered overbought: {cur_rsi:.1f} (was {prev_rsi:.1f})",
            'priority': 'medium',
        })
    return alerts
Python — Moving Average Crosses 
def check_ma_cross_alert(snap: dict) -> list:
    '''Detect golden and death crosses.'''
    alerts = []
    cur  = snap['cur']
    prev = snap['prev']

    if cur['sma50'] > cur['sma200'] and prev['sma50'] <= prev['sma200']:
        price = cur['close']
        alerts.append({
            'type': 'GOLDEN_CROSS',
            'msg':  f"🟡 {snap['ticker']} GOLDEN CROSS — SMA50 crossed above SMA200 | Price: ${price:.2f}",
            'priority': 'high',
        })
    if cur['sma50'] < cur['sma200'] and prev['sma50'] >= prev['sma200']:
        price = cur['close']
        alerts.append({
            'type': 'DEATH_CROSS',
            'msg':  f"💀 {snap['ticker']} DEATH CROSS — SMA50 crossed below SMA200 | Price: ${price:.2f}",
            'priority': 'high',
        })
    return alerts
Python — Price Crossing Moving Averages 
def check_price_ma_cross(snap: dict) -> list:
    '''Detect price crossing above/below key MAs.'''
    alerts = []
    cur  = snap['cur']
    prev = snap['prev']
    ticker = snap['ticker']

    for ma, label in [('sma20','SMA20'),('sma50','SMA50'),('sma200','SMA200')]:
        if cur['close'] > cur[ma] and prev['close'] <= prev[ma]:
            alerts.append({
                'type': f'PRICE_ABOVE_{ma.upper()}',
                'msg':  f'⬆ {ticker} price crossed ABOVE {label}: ${cur["close"]:.2f}',
                'priority': 'low' if ma == 'sma20' else 'medium',
            })
        if cur['close'] < cur[ma] and prev['close'] >= prev[ma]:
            alerts.append({
                'type': f'PRICE_BELOW_{ma.upper()}',
                'msg':  f'⬇ {ticker} price crossed BELOW {label}: ${cur["close"]:.2f}',
                'priority': 'low' if ma == 'sma20' else 'medium',
            })
    return alerts

def check_macd_cross(snap: dict) -> list:

    alerts = []
    cur  = snap['cur']
    prev = snap['prev']

    if cur['macd'] > cur['macd_s'] and prev['macd'] <= prev['macd_s']:

            'msg':  f"🟢 {snap['ticker']} MACD bullish crossover | Price: ${cur['close']:.2f}",

            'msg':  f"🔴 {snap['ticker']} MACD bearish crossover | Price: ${cur['close']:.2f}",
            'priority': 'medium',
        })
    return alerts

Alert Loop with Cooldown

Prevent spam: track which signals you've already alerted on, and skip re-alerting within a cooldown window (usually 1 day for daily bars):

Python 
# Prevent re-alerting on the same signal within a cooldown window
alert_history: dict[str, date] = {}  # key: 'TICKER_SIGNAL_TYPE'

def should_alert(ticker: str, signal_type: str, cooldown_days: int = 1) -> bool:
    '''Return True if we haven't already alerted on this signal today.'''
    key = f'{ticker}_{signal_type}'
    last = alert_history.get(key)
    if last is None or (date.today() - last).days >= cooldown_days:
        return True
    return False

def record_alert(ticker: str, signal_type: str) -> None:
    alert_history[f'{ticker}_{signal_type}'] = date.today()

def process_alerts(alerts: list, email_priorities: list = ['high']) -> None:
    '''Process a list of alert dicts — log all, email high priority.'''
    for alert in alerts:
        ticker, sig_type = alert['msg'].split()[1], alert['type']
        if not should_alert(ticker, sig_type):
            continue

        log_alert(alert['msg'])
        record_alert(ticker, sig_type)

        if alert['priority'] in email_priorities:
            send_email(
                subject=f"[OpenClaw Alert] {alert['type']} — {ticker}",
                body=alert['msg'] + f"\n\nGenerated by OpenClaw TA Alert System\n{datetime.now()}"
            )

def run_alert_loop(watchlist: list, check_interval: int = 3600) -> None:
    '''
    Poll watchlist every check_interval seconds.
    Default: once per hour (daily candles don't change more often).
    '''
    print(f'🔔 Alert loop started | {len(watchlist)} tickers | interval: {check_interval}s')

    while True:
        print(f"\n[{datetime.now().strftime('%H:%M:%S')}] Running checks...")

        for ticker in watchlist:
            snap = get_snapshot(ticker)
            if not snap:
                continue

            all_alerts = (
                check_rsi_alert(snap) +
                check_ma_cross_alert(snap) +
                check_price_ma_cross(snap) +
                check_macd_cross(snap)
            )
            process_alerts(all_alerts)

        print(f'  Next check in {check_interval // 60} minutes...')
        time.sleep(check_interval)

WATCHLIST = ['AAPL','MSFT','GOOGL','NVDA','SPY','QQQ','GLD','TLT']
run_alert_loop(WATCHLIST, check_interval=3600)

OpenClaw Daily Digest

Set alerts.daily_digest: true in ta-config.yaml to get a single end-of-day summary instead of individual real-time pings. Better for signal-rich markets where individual emails would be overwhelming.

Python 
def send_daily_digest(all_signals: list, cfg: dict) -> None:
    """
    Rolled-up digest of all signals fired during the day.
    Sent once at market close instead of individual alert emails.
    Configured via alerts.daily_digest in ta-config.yaml.
    """
    if not cfg["alerts"].get("daily_digest", False):
        return
    if not all_signals:
        print("No signals today — digest skipped.")
        return

    import smtplib
    from email.mime.text import MIMEText

    lines = [
        f"🦞 OpenClaw TA Daily Digest — {datetime.now().strftime('%Y-%m-%d')}",
        f"{'─'*45}",
        f"{len(all_signals)} signal(s) fired today:\n"
    ]
    for sig in all_signals:
        lines.append(
            f"  {sig['ticker']:<8} {sig['signal']:<20} "
            f"@ ${sig['price']:,.2f}  ({sig['time']})"
        )
    lines += [
        f"\n{'─'*45}",
        "Not financial advice. | learnopenclaw.org"
    ]

    body = "\n".join(lines)
    msg = MIMEText(body)
    msg["Subject"] = f"[OpenClaw] TA Digest — {len(all_signals)} signals — {datetime.now().strftime('%b %d')}"
    msg["From"]    = "openclaw@yourdomain.com"
    msg["To"]      = cfg["alerts"]["email"]

    with smtplib.SMTP("localhost") as s:
        s.send_message(msg)
    print(f"✓ Daily digest sent to {cfg['alerts']['email']}")

Alert Configuration Reference

Customize these parameters based on your trading style and alert preferences:

Parameter Description Recommended
oversold threshold RSI level to trigger buy watch 30
overbought threshold RSI level to trigger sell watch 70
cooldown_days Days before re-alerting same signal 1
email_priorities Which signals trigger immediate email ["high"]
