What is ClawHub?

The App Store for Your AI Agent

Think of ClawHub the way you think of the App Store or Chrome Extensions — except every "app" adds a new capability to your OpenClaw AI agent.

🧩 What is a skill?

A skill is a small package of instructions and tools that teaches OpenClaw how to do something new — like control Spotify, create Notion pages, or post to GitHub. Skills are installed in seconds and activated immediately.

👥 Who builds them?

Skills are built by the OpenClaw open-source community — developers, power users, and companies. Anyone can publish a skill to ClawHub. Quality varies, so always check the publisher and review count before installing.

💰 How much do they cost?

The vast majority of ClawHub skills are completely free and open-source. A small number of premium skills from verified publishers carry a one-time or subscription fee.

⚡ How do I install one?

Browse ClawHub from the OpenClaw dashboard or CLI, find a skill, review its permissions, and install with one command. OpenClaw will start using it immediately in your next conversation.

How it works

From Browse to Install in 4 Steps

Getting a new skill up and running takes under a minute.

1

Browse ClawHub

Search or filter skills by category in the OpenClaw dashboard or at clawhub.openclaw.ai.

2

Review Permissions

Every skill lists exactly what it can access — files, internet, messaging, shell. Read this carefully before proceeding.

3

Install the Skill

Click install or run the CLI command. OpenClaw downloads and verifies the skill package automatically.

4

Use It in Chat

The skill is live immediately. Just tell OpenClaw what you want — it knows the new skill is available.
