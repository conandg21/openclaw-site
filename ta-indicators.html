1. Setup & Data Fetch

Start by installing the required packages and fetching historical OHLCV data from Yahoo Finance. This foundation is used by all subsequent indicators.

python 
# pip install yfinance pandas-ta pandas
import yfinance as yf
import pandas_ta as ta
import pandas as pd

def get_ohlcv(ticker: str, period: str = "1y", interval: str = "1d") -> pd.DataFrame:
    """
    Fetch OHLCV from Yahoo Finance.
    period: 1d, 5d, 1mo, 3mo, 6mo, 1y, 2y, 5y, 10y, ytd, max
    interval: 1m, 5m, 15m, 30m, 1h, 1d, 1wk, 1mo
    Note: intraday intervals only available for last 60 days.
    """
    df = yf.download(ticker, period=period, interval=interval, progress=False)
    df.columns = [c.lower() for c in df.columns]
    df = df[["open","high","low","close","volume"]].dropna()
    print(f"{ticker}: {len(df)} bars | {df.index[0].date()} → {df.index[-1].date()}")
    return df

df = get_ohlcv("AAPL", period="1y")

The get_ohlcv() function handles downloading, normalizing column names, and validating data. Pass any ticker symbol (stocks, ETFs, crypto) and adjust the period and interval parameters to suit your analysis horizon.

2. Moving Averages — SMA and EMA

Simple Moving Averages (SMA) and Exponential Moving Averages (EMA) smooth price trends. Use SMA for longer-term trend analysis and EMA for reactive, recent-weighted signals.

python 
# Simple Moving Average — equal weight over N periods
df["sma_20"]  = ta.sma(df["close"], length=20)
df["sma_50"]  = ta.sma(df["close"], length=50)
df["sma_200"] = ta.sma(df["close"], length=200)

# Exponential Moving Average — more weight on recent prices
df["ema_12"] = ta.ema(df["close"], length=12)
df["ema_26"] = ta.ema(df["close"], length=26)

# Golden / Death Cross signal
df["golden_cross"] = (
    (df["sma_50"] > df["sma_200"]) &
    (df["sma_50"].shift(1) <= df["sma_200"].shift(1))
)
df["death_cross"] = (
    (df["sma_50"] < df["sma_200"]) &
    (df["sma_50"].shift(1) >= df["sma_200"].shift(1))
)

crosses = df[df["golden_cross"] | df["death_cross"]].copy()
for idx, row in crosses.iterrows():
    kind = "🟡 GOLDEN" if row["golden_cross"] else "💀 DEATH"
    print(f"{kind} CROSS: {idx.date()} | Price: ${row['close']:.2f}")

# Current MA status
latest = df.iloc[-1]
print(f"\nCurrent price: ${latest['close']:.2f}")
print(f"SMA20:  ${latest['sma_20']:.2f}  {'↑ above' if latest['close'] > latest['sma_20'] else '↓ below'}")
print(f"SMA50:  ${latest['sma_50']:.2f}  {'↑ above' if latest['close'] > latest['sma_50'] else '↓ below'}")
print(f"SMA200: ${latest['sma_200']:.2f} {'↑ above' if latest['close'] > latest['sma_200'] else '↓ below'}")
Golden Cross vs. Death Cross: A golden cross (50-day SMA crosses above 200-day SMA) often signals bullish momentum. A death cross indicates potential bearish reversal. These are useful for medium to long-term trend confirmation, not standalone trade signals.

3. RSI — Relative Strength Index

The Relative Strength Index measures momentum on a 0–100 scale. Values below 30 suggest oversold conditions (potential buy signals), while values above 70 suggest overbought conditions (potential sell signals).

python 
# RSI: measures momentum. 0-100 scale.
# < 30 = oversold (potential buy watch)
# > 70 = overbought (potential sell watch)
# Standard period: 14

df["rsi_14"] = ta.rsi(df["close"], length=14)

def classify_rsi(value: float) -> str:
    if value >= 80: return "Extremely Overbought 🔴"
    if value >= 70: return "Overbought ⚠️"
    if value >= 55: return "Bullish Momentum 🟢"
    if value >= 45: return "Neutral ⚪"
    if value >= 30: return "Bearish Momentum 🔵"
    if value >= 20: return "Oversold ⚠️"
    return "Extremely Oversold 🔴"

# Recent RSI readings
print("Recent RSI readings:")
for date, row in df[["close","rsi_14"]].tail(10).iterrows():
    label = classify_rsi(row["rsi_14"])
    print(f"  {date.date()}  RSI: {row['rsi_14']:5.1f}  {label}")

# RSI divergence hint — price making higher highs but RSI making lower highs
recent = df.tail(30)
price_trend = recent["close"].iloc[-1] - recent["close"].iloc[0]
rsi_trend = recent["rsi_14"].iloc[-1] - recent["rsi_14"].iloc[0]
if price_trend > 0 and rsi_trend < 0:
    print("\n⚠️  Potential bearish divergence: price rising, RSI falling")
elif price_trend < 0 and rsi_trend > 0:
    print("\n⚠️  Potential bullish divergence: price falling, RSI rising")

Divergence is key: When price makes higher highs but RSI makes lower highs, the uptrend may be losing strength. Conversely, if price makes lower lows while RSI makes higher lows, a reversal may be near.

4. MACD — Moving Average Convergence/Divergence

MACD combines two EMAs to identify momentum shifts. The histogram (MACD line minus signal line) crossing zero is a popular entry/exit signal.

python 
# MACD = EMA(12) - EMA(26)
# Signal = EMA(9) of MACD
# Histogram = MACD - Signal
# Crossover: MACD crosses above signal = bullish; below = bearish

macd = ta.macd(df["close"], fast=12, slow=26, signal=9)
df = pd.concat([df, macd], axis=1)

# Rename for clarity (pandas-ta uses dynamic column names)
macd_col   = [c for c in macd.columns if c.startswith("MACD_")][0]
signal_col = [c for c in macd.columns if c.startswith("MACDs_")][0]
hist_col   = [c for c in macd.columns if c.startswith("MACDh_")][0]

df["macd"]   = df[macd_col]
df["signal"] = df[signal_col]
df["hist"]   = df[hist_col]

# Detect crossovers
df["macd_bull"] = (df["macd"] > df["signal"]) & (df["macd"].shift(1) <= df["signal"].shift(1))
df["macd_bear"] = (df["macd"] < df["signal"]) & (df["macd"].shift(1) >= df["signal"].shift(1))

crossovers = df[df["macd_bull"] | df["macd_bear"]].tail(5)
for idx, row in crossovers.iterrows():
    kind = "🟢 BULL" if row["macd_bull"] else "🔴 BEAR"
    print(f"{kind} crossover: {idx.date()} | Price: ${row['close']:.2f} | MACD: {row['macd']:.4f}")
Reading MACD: When the histogram is positive (MACD above signal), momentum is bullish. When negative, momentum is bearish. Watch for zero crossings in the histogram as early momentum signals.

5. Bollinger Bands

Bollinger Bands envelope price within upper and lower bands (SMA ± 2 standard deviations). Narrow bands (squeeze) often precede breakouts; price near bands can signal mean reversion.

python 
# Bollinger Bands: SMA(20) ± 2 standard deviations
# Price near upper band = overbought relative to recent range
# Price near lower band = oversold relative to recent range
# Band squeeze (narrow bandwidth) often precedes a breakout

bb = ta.bbands(df["close"], length=20, std=2)
df = pd.concat([df, bb], axis=1)

# Extract columns
bb_upper = [c for c in bb.columns if "BBU" in c][0]
bb_mid   = [c for c in bb.columns if "BBM" in c][0]
bb_lower = [c for c in bb.columns if "BBL" in c][0]
bb_bw    = [c for c in bb.columns if "BBB" in c][0]  # bandwidth

df["bb_upper"] = df[bb_upper]
df["bb_mid"]   = df[bb_mid]
df["bb_lower"] = df[bb_lower]
df["bb_bw"]    = df[bb_bw]

# Detect squeeze (bandwidth in lowest 20th percentile of past year)
bw_20th = df["bb_bw"].quantile(0.20)
df["bb_squeeze"] = df["bb_bw"] < bw_20th

latest = df.iloc[-1]
print(f"Upper band: ${latest['bb_upper']:.2f}")
print(f"Mid (SMA20): ${latest['bb_mid']:.2f}")
print(f"Lower band: ${latest['bb_lower']:.2f}")
print(f"Bandwidth:  {latest['bb_bw']:.2f}")
print(f"Squeeze:    {'YES ⚡' if latest['bb_squeeze'] else 'No'}")

# %B — where is price within the bands (0 = at lower, 1 = at upper)
df["bb_pct"] = (df["close"] - df["bb_lower"]) / (df["bb_upper"] - df["bb_lower"])
print(f"%B:         {latest['bb_pct']:.2f} ({'Near upper band' if latest['bb_pct'] > 0.8 else 'Near lower band' if latest['bb_pct'] < 0.2 else 'Mid-range'})")

6. Full Indicator Stack — One Function

Combine all indicators into a single DataFrame for comprehensive analysis. Drop NaN rows to ensure all calculations are valid before interpretation.

python 
def build_indicator_stack(ticker: str, period: str = "1y") -> pd.DataFrame:
    """
    Returns a DataFrame with all core indicators computed.
    Drop NaN rows so all indicators have valid values.
    """
    df = get_ohlcv(ticker, period)

    # Trend
    df["sma_20"]  = ta.sma(df["close"], 20)
    df["sma_50"]  = ta.sma(df["close"], 50)
    df["sma_200"] = ta.sma(df["close"], 200)
    df["ema_12"]  = ta.ema(df["close"], 12)
    df["ema_26"]  = ta.ema(df["close"], 26)

    # Momentum
    df["rsi"]    = ta.rsi(df["close"], 14)
    df["stoch_k"] = ta.stoch(df["high"], df["low"], df["close"])["STOCHk_14_3_3"]

    # MACD
    macd = ta.macd(df["close"], 12, 26, 9)
    df["macd"]        = macd[[c for c in macd.columns if c.startswith("MACD_")][0]]
    df["macd_signal"] = macd[[c for c in macd.columns if c.startswith("MACDs_")][0]]
    df["macd_hist"]   = macd[[c for c in macd.columns if c.startswith("MACDh_")][0]]

    # Volatility
    bb = ta.bbands(df["close"], 20, 2)
    df["bb_upper"] = bb[[c for c in bb.columns if "BBU" in c][0]]
    df["bb_lower"] = bb[[c for c in bb.columns if "BBL" in c][0]]
    df["bb_bw"]    = bb[[c for c in bb.columns if "BBB" in c][0]]
    df["atr"]      = ta.atr(df["high"], df["low"], df["close"], 14)

    # Volume
    df["obv"] = ta.obv(df["close"], df["volume"])

    return df.dropna()

df = build_indicator_stack("AAPL", "1y")
print(f"Columns: {list(df.columns)}")
print(df.tail(3).to_string())
