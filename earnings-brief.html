What the Brief Delivers

The earnings brief is the capstone of Parts 1–4. It's a single pre-market email that shows you everything you need to know about earnings happening today and tomorrow. Here's what it looks like:

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 📋 OPENCLAW EARNINGS BRIEF — Wednesday Apr 30 2026 ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ TODAY'S REPORTS (your watchlist) ───────────────────────────────── MSFT → Reports TODAY (after close) EPS Estimate: $3.22 (range: $3.10–$3.38, 38 analysts) Beat Rate: 7/8 = 88% Avg EPS Surprise: +4.2% Avg Price Move: ±4.1% (3 up, 1 down) REPORTING TOMORROW ───────────────────────────────── AAPL → Reports TOMORROW (after close Thursday May 1) EPS Estimate: $1.61 Beat Rate: 8/8 = 100% Avg Price Move: ±3.0% (5 up, 1 down) ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ OpenClaw · learnopenclaw.org · Not financial advice ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

What Each Field Means

  • EPS Estimate: The consensus EPS projection from all analysts, plus the range (low–high) and total number of analyst forecasts.
  • Beat Rate: Historical percentage of earnings beats. "7/8 = 88%" means 7 of the last 8 reports beat consensus.
  • Avg EPS Surprise: Average percentage the company beat or missed by (positive = beat, negative = miss).
  • Avg Price Move: Average stock move (up or down) on the next trading day after earnings. Also shows the split (e.g., "3 up, 1 down").

The brief only shows earnings happening within your alert window (default: next 3 days). It's sent every weekday morning at 6:30 AM, so you can review before the market opens at 9:30 AM.

How It Combines All 5 Parts

The earnings brief is the payoff of the entire automation pipeline. Here's how each part feeds into it:

Part 1: Calendar

Knows which tickers on your watchlist report today and tomorrow. This determines who appears in the brief.

Part 2: Estimates

Pulls the current EPS and revenue consensus from yfinance, including analyst count and the range (low–high).

Part 3: Surprise

Reads the earnings_history.csv built by the surprise tracker. Calculates beat rate and average surprise percentage for each ticker.

Part 4: Price Reaction

Reads the price_reactions.csv to calculate average move magnitude and direction breakdown (how many ups vs. downs).

When you run the brief script at 6:30 AM, it simply fetches today's date, looks up which tickers report today/tomorrow, pulls their current estimates, and looks up their historical context from the CSVs built by Parts 3 and 4. All in seconds.

Full Configuration

Before running the brief script, you need the complete configuration file. This file ties together all 5 parts:

watchlist:
  tickers: [AAPL, MSFT, NVDA, AMZN, GOOGL]

calendar:
  lookahead_days: 5
  alert_days_before: 3
  market_hours_only: false

estimates:
  track_revisions: true
  revision_window_days: 30
  flag_revision_pct: 5.0
  sources: ["yfinance"]

surprise:
  log_history: true
  flag_threshold: 5.0
  csv_path: "data/earnings_history.csv"
  include_revenue_surprise: true

reaction:
  capture_window: "open_next_day"
  log_to_csv: true
  csv_path: "data/price_reactions.csv"
  compare_implied_move: true

alerts:
  email: "you@example.com"
  daily_digest: true
  send_time: "06:30"

scheduler:
  earnings_brief:
    script: earnings_brief.py
    schedule: "30 6 * * 1-5"

Save this as earnings-config.yaml in your OpenClaw working directory. Update the tickers list and email address to match your watchlist.

The Complete Script

This is the capstone script. It runs every weekday at 6:30 AM and checks for earnings happening today and tomorrow. It pulls estimates from yfinance, reads the history CSVs built by Parts 3 and 4, and formats everything into a clean pre-market brief.

"""
earnings_brief.py — OpenClaw Pre-Market Earnings Brief

The capstone script. Runs every weekday at 6:30 AM.
Checks for earnings today/tomorrow, pulls estimates,
loads historical beat rate + price reaction data,
and sends a clean pre-market email brief.

Requires: data/earnings_history.csv  (from earnings_surprise.py)
          data/price_reactions.csv   (from earnings_reaction.py)
          earnings-config.yaml
"""

import yaml
import yfinance as yf
import pandas as pd
from datetime import datetime, timedelta
import smtplib
from email.mime.text import MIMEText
import os

# ── Load config ────────────────────────────────────────────────────────────
with open("earnings-config.yaml") as f:
    cfg = yaml.safe_load(f)

tickers       = cfg["watchlist"]["tickers"]
lookahead     = cfg["calendar"]["lookahead_days"]
alert_window  = cfg["calendar"]["alert_days_before"]
alert_email   = cfg["alerts"]["email"]

today    = datetime.today().date()
tomorrow = today + timedelta(days=1)

# ── Load historical data (built by earlier scripts) ────────────────────────
def load_surprise_history():
    """Load the earnings beat/miss history CSV built by Part 3."""
    path = cfg["surprise"]["csv_path"]
    if os.path.exists(path):
        return pd.read_csv(path)
    return pd.DataFrame()

def load_reaction_history():
    """Load the price reaction history CSV built by Part 4."""
    path = cfg["reaction"]["csv_path"]
    if os.path.exists(path):
        return pd.read_csv(path)
    return pd.DataFrame()

surprise_df = load_surprise_history()
reaction_df = load_reaction_history()

# ── Helper: get beat rate for a ticker ────────────────────────────────────
def get_beat_rate(ticker):
    """Calculate historical beat rate from surprise CSV."""
    if surprise_df.empty:
        return None, None

    rows = surprise_df[surprise_df["Ticker"] == ticker]
    if rows.empty:
        return None, None

    beats = (rows["Result"] == "BEAT").sum()
    total = len(rows)
    rate  = round(beats / total * 100) if total > 0 else None
    avg_surprise = round(rows["Surprise %"].mean(), 1)
    return f"{beats}/{total} = {rate}%", avg_surprise

# ── Helper: get average price move for a ticker ───────────────────────────
def get_avg_move(ticker):
    """Calculate average price reaction magnitude from reaction CSV."""
    if reaction_df.empty:
        return None, None, None

    rows = reaction_df[reaction_df["Ticker"] == ticker]
    if rows.empty:
        return None, None, None

    avg_move = round(rows["Move %"].abs().mean(), 1)
    up_count = (rows["Direction"] == "UP").sum()
    dn_count = (rows["Direction"] == "DOWN").sum()
    return avg_move, up_count, dn_count

# ── Check upcoming earnings on watchlist ──────────────────────────────────
print(f"{'='*52}")
print(f"  📋 OPENCLAW EARNINGS BRIEF — {today.strftime('%A %b %d %Y')}")
print(f"{'='*52}\n")

brief_sections = []
today_reports  = []
soon_reports   = []

for ticker in tickers:
    stock = yf.Ticker(ticker)

    try:
        cal = stock.earnings_dates
        if cal is None or cal.empty:
            continue

        # Look at upcoming dates only
        upcoming = cal[cal.index.date >= today].head(1)
        if upcoming.empty:
            continue

        report_date = upcoming.index[0].date()
        days_away   = (report_date - today).days

        if days_away > alert_window:
            continue

        # Get estimates
        estimates = stock.earnings_estimate
        eps_est   = "N/A"
        eps_low   = "N/A"
        eps_high  = "N/A"
        analysts  = "N/A"

        if estimates is not None and not estimates.empty:
            row      = estimates.iloc[0]
            eps_est  = row.get("avg",             "N/A")
            eps_low  = row.get("low",             "N/A")
            eps_high = row.get("high",            "N/A")
            analysts = row.get("numberOfAnalysts","N/A")

        # Get historical context
        beat_str, avg_surprise = get_beat_rate(ticker)
        avg_move, up_ct, dn_ct = get_avg_move(ticker)

        entry = {
            "ticker":      ticker,
            "date":        report_date,
            "days_away":   days_away,
            "eps_est":     eps_est,
            "eps_low":     eps_low,
            "eps_high":    eps_high,
            "analysts":    analysts,
            "beat_str":    beat_str or "No history yet",
            "avg_surprise":avg_surprise,
            "avg_move":    avg_move,
            "up_ct":       up_ct,
            "dn_ct":       dn_ct,
        }

        if days_away == 0:
            today_reports.append(entry)
        else:
            soon_reports.append(entry)

    except Exception as e:
        print(f"  {ticker}: Error — {e}")

# ── Format the brief ──────────────────────────────────────────────────────
def format_entry(e):
    timing = "TODAY (after close)" if e["days_away"] == 0 else f"in {e['days_away']} day(s) — {e['date']}"
    lines  = [
        f"  {e['ticker']}  →  Reports {timing}",
        f"     EPS Estimate:     ${e['eps_est']}  (range: ${e['eps_low']}–${e['eps_high']}, {e['analysts']} analysts)",
        f"     Beat Rate:        {e['beat_str']}",
    ]
    if e["avg_surprise"]:
        lines.append(f"     Avg EPS Surprise: +{e['avg_surprise']}%")
    if e["avg_move"]:
        lines.append(f"     Avg Price Move:   ±{e['avg_move']}%  ({e['up_ct']} up, {e['dn_ct']} down)")
    return "\n".join(lines)

brief_text = f"{'='*52}\n"
brief_text += f"  📋 OPENCLAW EARNINGS BRIEF — {today.strftime('%A %b %d %Y')}\n"
brief_text += f"{'='*52}\n\n"

if today_reports:
    brief_text += "  TODAY'S REPORTS (your watchlist)\n"
    brief_text += "  " + "─"*40 + "\n"
    for e in today_reports:
        brief_text += format_entry(e) + "\n\n"

if soon_reports:
    brief_text += "  REPORTING SOON\n"
    brief_text += "  " + "─"*40 + "\n"
    for e in soon_reports:
        brief_text += format_entry(e) + "\n\n"

if not today_reports and not soon_reports:
    brief_text += "  ✅ No earnings on your watchlist in the next few days.\n\n"

brief_text += f"{'='*52}\n"
brief_text += "  OpenClaw · learnopenclaw.org · Not financial advice\n"
brief_text += f"{'='*52}\n"

# ── Print to terminal ─────────────────────────────────────────────────────
print(brief_text)

# ── Send email ────────────────────────────────────────────────────────────
if today_reports or soon_reports:
    msg = MIMEText(brief_text)
    msg["Subject"] = f"OpenClaw Earnings Brief — {today.strftime('%b %d')}"
    msg["From"]    = alert_email
    msg["To"]      = alert_email

    # Uncomment and configure your SMTP settings:
    # with smtplib.SMTP("smtp.gmail.com", 587) as server:
    #     server.starttls()
    #     server.login("your@gmail.com", "your_app_password")
    #     server.send_message(msg)

    print(f"\n📧 Brief ready to send to: {alert_email}")
    print("   (Configure SMTP settings above to enable sending)")

Paste this script as earnings_brief.py in your OpenClaw working directory. The script prints the brief to your terminal every time it runs, and includes commented-out SMTP code for email sending (see the FAQ below for Gmail setup).

Setting Up the Daily Schedule

Once your script is ready, add a cron job to run it every weekday at 6:30 AM:

30 6 * * 1-5 cd /path/to/openclaw && python earnings_brief.py

Step-by-Step Setup

  1. Install dependencies: pip install yfinance pyyaml pandas
  2. Create the directory structure: 
    mkdir -p data
cp earnings-config.yaml ./
  3. Save the script as earnings_brief.py
  4. Test the script manually: 
    python earnings_brief.py
  5. If it works, add the cron job: crontab -e and paste the line above.
  6. Verify the cron was added: crontab -l

That's it. Your brief will arrive every weekday morning at 6:30 AM (or whenever you set it).

FAQ

Q: How long until the brief is actually useful?
The calendar and estimates work from day one. But the beat rate and average move sections need historical data. Run Parts 3 and 4 for 2–3 months (roughly 8–12 earnings cycles) to build meaningful datasets. After that, the brief context is gold.
Q: Does this work for any stock?
Any stock covered by yfinance. Works best for large-cap stocks with many analyst estimates (20+). Small-cap stocks may not have earnings date data or consensus estimates. Always test your ticker first: yf.Ticker("TICK").earnings_dates
Q: Can I use Gmail to send the emails?
Yes. Gmail blocks direct login in scripts for security reasons, but supports "App Passwords" for automation. Enable 2FA on your Google account, then generate an app-specific password in your Security settings. Use that password in the SMTP code (not your regular Gmail password). Google's app password guide.
