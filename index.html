What is it?

OpenClaw: From Chatbot to Agent

OpenClaw is a free, open-source autonomous AI agent developed by Peter Steinberger. Unlike a chatbot that just answers questions, OpenClaw can execute tasks — on your behalf, on your machine.

🧠 What it is

An autonomous AI agent that lives on your computer. It uses large language models (like Claude or GPT) as its brain, and acts as a personal assistant that can actually do things — not just talk about them.

📱 How you talk to it

You interact with OpenClaw through everyday messaging apps you already use — WhatsApp, Telegram, Slack, Discord, or iMessage. Just send a message; the agent handles the rest.

🔓 Why it's different

It's fully open-source, runs locally on your device, and is completely free. No subscription needed — just bring your own API key for your preferred AI model, or run a local model entirely offline.

📖 A little history

Launched in November 2025 as Clawdbot by Austrian developer Peter Steinberger. After trademark issues it became Moltbot, then OpenClaw in early 2026 — and promptly hit 60,000 GitHub stars in 72 hours.

How to use it

Get Up and Running

OpenClaw is designed to be set up once and then used entirely through your favourite chat app. Here's how to get started.

1

Install OpenClaw

Clone or download the project from GitHub. Make sure you have Node.js installed on your machine.

git clone https://github.com/OpenClaw/openclaw
2

Add your API key

OpenClaw is model-agnostic — use Claude, GPT-4, or a local model. Add your API key to the .env config file. No recurring subscription needed.

3

Connect your messaging app

Link OpenClaw to your preferred messaging platform — WhatsApp, Telegram, Slack, Discord, or iMessage. Follow the setup guide in the official docs to authenticate the connection.

4

Install Agent Skills (optional)

Browse ClawHub, OpenClaw's skill marketplace, to add capabilities — from calendar management to web automation. Over 100 skills are available. Always review skill permissions carefully before installing.

5

Start giving commands

Open your messaging app and tell OpenClaw what you need. It will reason through the task and take action. Try:

"Summarise my emails from today"
"Schedule a meeting with John for Thursday at 2pm"
"Search the web for the latest AI news and send me a summary"
⚠️

Security note: OpenClaw has experienced security incidents — malicious skills distributed via ClawHub installed malware on users' machines. Only install skills from trusted sources, review all permissions carefully, and consider running OpenClaw inside an isolated environment (e.g. a virtual machine) for extra safety.

Features & Benefits

What OpenClaw Can Do For You

OpenClaw isn't just a text generator. It's a true task executor — bridging the gap between AI intelligence and real-world action.

💻

Execute Shell Commands

Runs commands on your machine on your behalf — automate system tasks, scripts, and processes with plain-language instructions.

📁

Read & Write Files

Create, edit, move, and organise files on your computer. Ask it to draft a document, summarise a PDF, or reorganise a folder.

🌐

Browse the Web

Search the internet, extract information from websites, and compile research — all triggered by a simple chat message.

📧

Send Emails

Draft and send emails on your behalf. Summarise your inbox, reply to threads, or set up automated email workflows.

📅

Manage Your Calendar

Schedule meetings, set reminders, check availability, and manage your calendar through natural conversation.

🔒

Local & Private

Runs entirely on your own machine. Your data stays with you — no cloud processing of personal files unless you choose it.

🤖

Model Agnostic

Works with Claude, GPT-4, Gemini, or fully local models like LLaMA. Switch models anytime with a single config change.

🧩

100+ Agent Skills

Extend capabilities with skills from ClawHub — web automation, Slack integration, coding helpers, and much more.

💬

Works in Your Chat App

No new interface to learn. Use WhatsApp, Telegram, Slack, Discord, or iMessage — wherever you already are.

🆓

Completely Free

Open-source and free forever. Bring your own API key — no recurring subscription, no vendor lock-in.
